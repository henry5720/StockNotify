# frontend

## 1. add package
``` bash
# react
yarn add react react-dom
# babel
yarn add @babel/core @babel/preset-env @babel/preset-react babel-loader @babel/eslint-parser @babel/plugin-proposal-class-properties
# webpack
yarn add webpack webpack-cli webpack-dev-server
# eslint+prettier
yarn add eslint prettier eslint-plugin-prettier eslint-config-prettier  eslint-plugin-json eslint-plugin-react --dev
```
